## 🔍 Loyiha Haqida
**Pro-Crypto Terminal** — bu shunchaki narx kuzatuvchi dastur emas, balki professional treyderlar uchun mo'ljallangan **Fintech Dashboard**. U Binance API yordamida real vaqtdagi bozor dinamikasini tahlil qiladi va murakkab texnik indikatorlarni vizuallashitiradi.

> **Maqsad:** Arbitraj imkoniyatlarini ko'rish, bozor trendini tahlil qilish va AI-ga asoslangan trading algoritmlari uchun poydevor yaratish.

---

## 🔥 Asosiy Xususiyatlari

| Funktsiya | Tavsif |
| :--- | :--- |
| **🚀 Real-time Data** | Binance WebSocket/REST API orqali soniyalik yangilanish. |
| **📊 Advanced Charting** | Plotly asosidagi interaktiv Candlestick va hajmlar grafigi. |
| **🛠 Tech Indicators** | EMA (9, 21), RSI (14) va MACD integratsiyasi. |
| **🏦 Order Book** | Bozor chuqurligini (Bids/Asks) ko'rsatuvchi real-vaqt jadvali. |
| **🎨 Professional UI** | Dark Mode dizayn, mobil qurilmalarga moslashuvchan interfeys. |
